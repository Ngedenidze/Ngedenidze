# About Me

I build and scale product systems end-to-end across web and backend, with a focus on performance, data architecture, and real-world execution.

As a founding engineer, I’ve taken products from zero to production. At Fixify, I built and scaled an automotive marketplace handling 5,000+ monthly service requests and a network of 150+ providers. At Synapse Studio, I led architecture and implementation across multiple early-stage products.

My work centers on designing systems that hold under growth. Backend architecture with GraphQL, Prisma, and PostgreSQL, paired with structured frontends in React and Next.js. I focus on data modeling, API design, and building products that are fast, predictable, and operationally reliable.

I also have a background in machine learning research, where I optimized large-scale CNN training workflows and published work on reducing backpropagation cost. I’m interested in applying that kind of efficiency thinking to production systems.
