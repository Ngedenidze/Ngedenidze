# About Me

I build operational software systems across backend, frontend, and data infrastructure, with a focus on performance, reliability, and real-world execution.

As a founding engineer, I’ve taken products from zero to production, from scaling Fixify’s marketplace to building early-stage systems at Synapse Studio. My work centers on designing systems that hold under growth, spanning backend architecture, structured frontends, offline mobile workflows, and operational data systems.

I also have a background in machine learning research, particularly large-scale training efficiency, which continues to influence how I think about production systems.

## Current Work

### Vellin — AI-Powered Restaurant Inventory & Operations Platform

Founding Engineer and CTO building the platform end-to-end across backend, web, and mobile.

I design and implement both the frontend and backend, including:

* Architecting and building the multi-tenant .NET + PostgreSQL backend
* Developing the React web application and Expo mobile application
* Designing offline-first inventory counting and sync workflows
* Building vendor ordering, purchasing, and operational analytics systems
* Implementing realtime and event-driven infrastructure for operational workflows

Current focus areas include:

* Inventory and purchasing automation
* Offline synchronization and idempotent write handling
* Permission systems and multi-role access control
* Invoice ingestion and structured operational data workflows
